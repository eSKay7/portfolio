A fully responsive personal portfolio built with React + Vite, designed to showcase my projects, experience, and technical work.
Features a clean UI, smooth transitions, dynamic content panels, and custom-built components.

Tech Stack
React 19
Vite 7
JavaScript (ES2025)
CSS
GitHub Pages (Deployment)

Features
Interactive project cards with expandable sections
Smooth panel transitions (degree/awards section)
Fully responsive layout
Automatic deploy pipeline via GitHub Pages

Installation
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>
npm install
npm run dev

Build
npm run build

Deploy
Deployment uses gh-pages:
npm run deploy

This builds the site and pushes the dist/ folder to the gh-pages branch automatically.
